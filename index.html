What is this website?

This website is an independent public documentation archive. It preserves and organizes publicly available open-source information related to reported events for journalistic, research, and historical reference.

How to read this site:
Each “Case” represents one documented event. Each case page includes a clear explanation and links to original public sources so readers can independently review the information.
